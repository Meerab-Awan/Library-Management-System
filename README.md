# 📚 Library Management System (C++)

A robust, console-based management application built in C++ that simulates a real-world library database. This project was developed as a major milestone to demonstrate core programming logic, user workflows, and data persistence.

---

## 🌟 The Story Behind the Project
Transitioning from a pre-medical academic background into Computer Science has been an incredibly rewarding challenge. This project represents my first major milestone in software development, translating complex logic, syntax, and debugging hours into a fully functional, real-world application. It stands as a testament to my dedication to learning programming fundamentals from scratch.

---

## 🚀 Key Features
* *Full CRUD Operations:* Add new books, search existing inventory, modify book details, and delete outdated records.
* *Persistent Storage (File Handling):* Uses C++ file streams (fstream) to read and write database records directly to local text files, ensuring no data is lost when the program closes.
* *Student Activity Logs:* Tracks student borrowing records, monitors issue dates, and manages book returns seamlessly.
* *Input Validation:* Built with clean logic to handle incorrect user inputs without crashing the system.

---

## 🛠️ Tech Stack & Concepts Applied
* *Language:* C++
* *File I/O:* ifstream, ofstream, and fstream for text-based database management.
* *Data Structures:* Structs and Arrays to organize book and student records.
* *Core Logic:* Control structures, loops, functions, and file pointer manipulation.

---

## 💻 How It Works
When the program runs, it displays an interactive main menu:
1. *Add Book Record* (Saves directly to database file)
2. *Show All Books* (Reads and formats data from the file)
3. *Search Book* (Fast lookup by Book ID)
4. *Issue Book* (Links a book ID to a Student ID)
5. *Return Book* (Clears active issue logs)
